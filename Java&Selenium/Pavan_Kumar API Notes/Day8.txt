--------
SOAPUI
----------

Advantages
---------
1) Functional testing
2) Security testing
3) Performance/Load testing
4) Supports both SOAP & Rest services

How to install SOAPUI
-------
Download link: https://www.soapui.org/downloads/soapui.html

Create a SOAP Project, Test Suite, Test Case
-----------------------------
WSDL: Web Service Description Language

https://www.dataaccess.com/webservicesserver/NumberConversion.wso?WSDL
https://www.dataaccess.com/webservicesserver/TextCasing.wso?WSDL
https://footballpool.dataaccess.eu/info.wso?WSDL

-> Create SOAP Project using WSDL
-> Create test suite & Test cases
-> Added contains assertion
-> Execute tests in sequence & parallel
-> Documentation


Create a Rest Project, Test Suite, Test Case
---------------------------------
